# UGC Portfolio

O portfolio é o único proof of work necessário para fechar primeiros clientes. Não precisas de clientes para criar um portfolio.

## Spec Content (portfolio sem clientes)

**O que é:** vídeos criados com produtos próprios, filmados e editados como se fossem anúncios reais para uma brand.

**Como fazer:**
1. Escolher 3-5 produtos que já tens (skincare, wellness, tech, home)
2. Filmar como se fosses um creator contratado pela brand
3. Editar com estilo de anúncio Meta/TikTok (hook nos primeiros 3s, CTA no final)
4. Não pedir permissão à brand — é spec, não oficial

## O que incluir no portfolio

| Elemento | Porquê |
|----------|--------|
| 3-5 vídeos de nichos diferentes | Mostra versatilidade |
| Bio com niche + content style | Brand avalia rapidamente se faz sentido |
| Serviços e preços (ou "from €X") | Filtra leads não qualificados |
| Contacto directo | Reduz fricção |
| Testimonial (se tiveres) | Social proof, mesmo que seja 1 |

## Formatos a incluir

- Testimonial style (creator fala sobre produto)
- Unboxing
- Tutorial / how-to
- Aesthetic product shot + VO

## Onde hospedar

| Plataforma | Custo | Ideal para |
|-----------|-------|-----------|
| Carrd | ~€19/ano | Simplicidade, look profissional rápido |
| Notion | Grátis | Mais flexível, menos profissional visualmente |
| Site próprio | €100-300/ano | Escalabilidade mas overhead |

**Recomendação Viralto:** Carrd para arrancar, migrar para site próprio após primeiro cliente.
