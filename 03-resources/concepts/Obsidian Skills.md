# Obsidian Skills

Claude Code skills for Obsidian vault integration. Installed via claude-obsidian plugin.

## Installed Skills (15)

- `/wiki-lint` — check vault health, dead links, missing frontmatter
- `/autoresearch` — research a topic, file findings into vault
- `/save` — save current conversation as a vault note
- `/wiki-mode` — set vault methodology (PARA, LYT, Zettelkasten, Generic)
- `/wiki-ingest` — ingest a source into the vault
- `/wiki-query` — query vault knowledge
- `/think` — structured thinking framework
