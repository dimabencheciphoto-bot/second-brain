# UGC Agency Blueprint — startaugcagency.com

## Key Claims
- Portfolio spec (3-5 vídeos com produtos próprios) é o passo 1 antes de qualquer outreach
- Target ideal: DTC Shopify small-mid, mais abertos a testar UGC novo
- Primeiro cliente possível em 2-4 semanas com outreach focado (20-30 brands/semana)
- Criar script templates reutilizáveis por formato (review, tutorial, unboxing)

## Contribuição para o tema
Fornece o blueprint estrutural para o lançamento de uma agência UGC do zero.
